Задание 2
Создайте полноценный файловый менеджер с использованием всех пройденных тем.

Требования:
Создайте проект fullFileManager со следующей структурой:
server.js – основной HTTP сервер
fileEvents.js – модуль с EventEmitter и функциями для работы с файлами
logger.js – модуль для логирования событий

В модуле fileEvents.js:
Создайте EventEmitter

Реализуйте функции: readFileAsync, writeFileAsync, deleteFileAsync, copyFileWithStreams, getFileInfo
Каждая функция генерирует события: start, success, error

В модуле logger.js:
Импортируйте EventEmitter из fileEvents.js
Зарегистрируйте обработчики для всех событий и выводите их в консоль с временными метками

В файле server.js:
Импортируйте модули http, url, а также ваши модули fileEvents и logger
Создайте HTTP сервер с маршрутами для каждой операции
Используйте потоки (streams) для копирования больших файлов
Запустите сервер на порту 3333
